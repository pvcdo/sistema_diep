Criação sistema diep

Server

Referência em DEV.

Na pasta server iniciamos o projeto node

yarn init -y

Depois instalamos o express e o seu módulo de tipos

yarn add express yarn add -D @types/express

Instalamos o typescript como dependência de desenvolvimento

yarn add -D typescript

Instala o pacote ts-node pra conseguir executar o typescript

yarn add -D ts-node

Criamos o arquivo de configurações do typescript

yarn tsc --init

Depois de configurado o express com typescript vamos instalar as demais dependências do projeto

yarn add bcrypt cookie-parser cors jsonwebtoken mongoose nodemon
  • bcrypt = criptografia de senhas
  • cookie-parser =
  • cors = para podermos mandar requisições para o nosso mesmo domínio. Quando lidamos com front e back há conflitos que este módulo trata
  • jsonwebtoken = autenticação de usuários
  • mogoose = acesso ao banco de dados mongo
  • nodemon = servidor que escuta as alterações e reinicia o app a cada salvamento de versão
